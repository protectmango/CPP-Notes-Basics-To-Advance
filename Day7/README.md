# Class & Object

**Class** is a collection of data member and member function.

To create a class we use keyword `class.`

**Syntax**
```c++
class Student{
    int rollno;
    void func(){

    }
};
```
|||
|---|---|
|class|`keyword`|
|**Student**|`tagname`|
|**int rollno;**|`attribute`| 
|void func(){}|`method`|
|||
